# Matrix Multiplication

Bu proje, iki matrisin çarpımını C# kullanarak gerçekleştiren bir console uygulamasıdır. Projede iki farklı yöntem kullanılmıştır:  

1. **Normal (Iterative) Multiplication**  
   - Geleneksel yöntemle matris çarpımı yapılır.  
   - İki for döngüsü içinde satır ve sütunlar üzerinden iterasyon ile çarpım hesaplanır.  
   
2. **Recursive Multiplication**  
   - Matris çarpımını özyinelemeli (recursive) yöntemle hesaplar.  
   - Büyük matrisleri daha küçük alt matrislere bölerek çarpma işlemini gerçekleştirir.  

## Kullanım
- Uygulama console üzerinden çalıştırılır.
- Kullanıcıdan matris boyutları ve elemanları alınır.
- Seçilen yöntemle matris çarpımı ekrana yazdırılır.
